O que é CSS?

Definição

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar páginas HTML. Ele define toda a parte visual de um site, como cores, tamanhos, fontes, espaçamento e posicionamento dos elementos.

Sem o CSS, os sites seriam apenas textos simples sem organização visual. O CSS transforma páginas básicas em interfaces modernas, bonitas e fáceis de usar.

Ele trabalha junto com o HTML, que é responsável pela estrutura do conteúdo.

Seletores

Os seletores são usados para escolher quais elementos do HTML receberão estilos. Eles funcionam como "alvos" dentro do código CSS.

Existem vários tipos de seletores, como:

• Seletor de elemento (ex: h1, p)
• Seletor de classe (ex: .caixa)
• Seletor de id (ex: #menu)

Exemplo: h1 { color: white; } → todos os títulos h1 ficam brancos.

Propriedades

As propriedades são responsáveis por definir o que será modificado no elemento. Elas controlam todos os estilos aplicados na página.

Exemplos importantes:

• color → muda a cor do texto
• font-size → muda o tamanho da letra
• background-color → muda o fundo
• margin → cria espaçamento externo
• padding → cria espaçamento interno

Valores

Os valores definem como cada propriedade será aplicada. Eles são o resultado final do estilo no site.

Exemplos:

color: white; → texto branco
font-size: 20px; → letra maior
background-color: black; → fundo preto

Sem valores, o CSS não funciona corretamente.

Estrutura do CSS

O CSS segue sempre o mesmo padrão de escrita:

seletor { propriedade: valor; }

Exemplo real:

h1 { color: lightblue; font-size: 30px; }

Isso significa que todos os títulos h1 ficarão com cor azul clara e tamanho maior.

Conclusão

O CSS é essencial para o desenvolvimento de sites modernos. Ele permite transformar páginas simples em sites organizados, bonitos e profissionais.

Aprender CSS é um passo importante para quem quer trabalhar com tecnologia e criação de sites.
